HTML Tags Cheat Sheet

Headings

Heading tags from `h1` to `h6` lets you add headings to your page. There should be only be one `h1` tag on your page, for SEO reasons.

Paragraphs

Paragraphs should be added using the `p` tags.

Lists

Unordered Lists

Unordered lists are lists without indexes.

Types of lists

  • Ordered Lists `ol`
  • Unordered Lists `ul`

Ordered Lists

Ordered lists are lists with index

Anchor Element

Anchor elements `a` are used to anchor some text to urls on the web. In other words, they are used to create links.Go to DuckDuckGo to search for more information

the `_target` attribute can be used to open the link in a new tab.This link will open in a new tab

Text Emphasis

Italic

The `em` tag is used to add emphasis to text, this is usually interpreted as italic by browsers.

Bold

The `strong` tag is used to add importance or urgency to text, this is usually interpreted as bold by browsers.

Organizing content with `div` and `span`

`div`

The `div` tag is a block element, so each element shows up on a different line.
